# Project: Dental Scribe AI

## Clinical Standards
- *SOAP Format:* Always structure clinical notes into Subjective, Objective, Assessment, and Plan.
- *Tooth Numbering:* Always use the *Universal Numbering System* (Teeth 1-32 for adults).
- *Surfaces:* Use standard abbreviations: M, O, D, L, B, F.
- *Compliance:* Adhere to HIPAA standards. Never log PII (Personally Identifiable Information).
Always use professional medical shorthand in the SOAP note (e.g., pt for patient, c/o for complains of, WNL for within normal limits).

## Technical Standards
- *Framework:* Next.js 15+ (App Router).
- *Language:* TypeScript (Strict mode).
- *Styling:* Tailwind CSS.
- *Voice-to-Text:* Integration with OpenAI Whisper-1.
